# weather-speech-bot
Бот телеграм который определяет погоду по местоположению и проговаривает её

# Установка

<code>
pip install beautifulsoup4
  </code>
<code>
pip install pytelegrambotapi
  </code>
<code>
pip install requests
  </code>
<code>
nano config.py
  </code>
  
Написать
<code>
token = 'your_token'
  </code>

# Запуск
<code>
python main.py
  </code>
