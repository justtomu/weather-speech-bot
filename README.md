# weather-speech-bot
Бот телеграм который определяет погоду по местоположению и проговаривает её

# Установка

<code>
pip install beautifulsoup4
  </code>
  <br />
<code>
pip install pytelegrambotapi
  </code>
  <br />
<code>
pip install requests
  </code>
  <br />
<code>
nano config.py
  </code>
  <br />
Написать
<code>
token = 'your_token'
  </code>

# Запуск
<code>
python main.py
  </code>
